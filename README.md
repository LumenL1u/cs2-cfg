# CS2 个人配置与跑图脚本使用说明

本仓库/文件夹包含了适用于《反恐精英 2》(CS2) 的个人优化配置及专业的跑图练习脚本。通过加载这些配置，你可以优化游戏性能、统一键位绑定，并获得极佳的单人或完美平台跑图体验。

## 启动项参考
```
-perfectworld -novid -high -nojoy -allow_third_party_software +exec autoexec.cfg
```

## 📁 包含文件

- **`autoexec.cfg`**: 核心自动执行文件，包含网络优化、UI 调整、一键购买及日常竞技键位绑定。
- **`practice.cfg`**: 专属跑图配置文件，提供无限道具、防具、控制 BOT 及切刀等进阶功能。
- **`practice_help.cfg`**: 跑图帮助文档，在游戏内控制台提供可视化的快捷键说明。

---

## 🚀 安装与使用方法

1. **放置文件**: 将以上三个 `.cfg` 文件放入你的 CS2 游戏配置文件夹中，路径通常为：
   `Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg`

2. **启动项设置**: 在 Steam 库中右键 CS2 -> 属性 -> 启动选项，添加 `+exec autoexec.cfg` 确保每次进游戏自动加载。

3. **日常竞技**: 启动游戏后，`autoexec.cfg` 会自动生效，你可以在控制台输入 `cfg` 随时重新加载。

4. **开始跑图**: 进入单人离线地图或完美平台跑图服务器后，在控制台输入 `pt` 即可加载完整跑图环境。输入 `ph` 可在控制台调出按键帮助菜单。

## ⚙️ 日常竞技快捷键 (`autoexec.cfg`)

以下快捷键在你启动游戏后默认生效：

### 一键购买与投掷物

| 按键 | 功能说明 |
| :--- | :--- |
| **F1** | 购买 AK-47 / M4A1 + 全甲 |
| **F2** | 购买 全套投掷物 (烟、闪、火、雷) + 全甲 |
| **F3** | 购买 AWP + 全甲 |
| **Z** | 切换至 高爆手雷 |
| **X** | 切换至 闪光震撼弹 |
| **C** | 切换至 烟雾弹 |
| **V** | 切换至 燃烧瓶/燃烧弹 |

### 战术与实用操作

| 按键 | 功能说明 |
| :--- | :--- |
| **MOUSE4** (侧键) | 切换左右手模型 |
| **N** | 一键跳投 (精准投掷) |
| **SPACE** / **滚轮下** | 跳跃 |
| **Caps Lock** | 切换雷达缩放比例 |
| **F4** | 队伍频道发送发枪请求 (`.drop`) |
| **F5** | 无线电消息 |
| **F7** / **F8** | 队伍频道发送暂停/取消暂停 (`.pause` / `.unpause`) |

## 🎯 跑图专属快捷键 (`practice.cfg`)

在控制台输入 `pt` 加载跑图配置后，以下所有作弊指令及跑图快捷键将生效。配置已默认开启 `sv_cheats 1`，并设置无限子弹、60分钟对局及初始99999金钱。

### 基础跑图与功能

| 按键 | 功能说明 |
| :--- | :--- |
| **Alt** | 飞行穿墙 (noclip) |
| **F1** | 重复上一个投掷物 |
| **F2** | 一键消除场上所有投掷物效果 (烟/火/闪) |
| **F3** | 切换显示本地弹着点开关 |
| **F9** | 补充护甲 |
| **\** | 屏蔽闪光效果 (完美平台跑图适用) |
| **Enter** | 回到上一次投掷的位置 (完美平台跑图适用) |

### 趣味换刀功能

| 按键 | 切换武器 |
| :--- | :--- |
| **F5** / **kp_9** | 一键换爪子刀 |
| **F6** | 一键换 M9 刺刀 |
| **F7** | 一键换骷髅匕首 |
| **F8** | 一键换蝴蝶刀 |

### BOT 控制系统

| 按键 | 功能说明 |
| :--- | :--- |
| **-** | 添加一个 T 阵营 BOT 并放置在准星处 |
| **=** | 添加一个 CT 阵营 BOT 并放置在准星处 |
| **[** | 添加一个蹲着的 BOT (完美平台跑图适用) |
| **]** | 切换场上 BOT 的 蹲下/站立 状态 |
| **Backspace** | 一键踢出场上所有 BOT |

### 进阶测试 (小键盘 Numpad)

| 按键 | 功能说明 |
| :--- | :--- |
| **kp_0** | 切换准星透明度 (用于练习屏幕中心感) |
| **kp_1** | 切换伤害倍率 (1倍 / 0.15倍，用于练习压枪) |
| **kp_3** | 切换 BOT 控制模式 (停止控制 / 同步移动 / 只控制 BOT) |
| **kp_4** | 控制 BOT 时，切换其移动方向 (同向 / 反向) |
| **kp_5** / **kp_6** | 切换 第一人称 / 第三人称视角 |
| **kp_7** / **kp_8** | 连跳功能 关闭 / 开启 |
